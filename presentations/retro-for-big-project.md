# Ретроспектива для великого проекту

Ми подивимось на те, що пішло не так, з точки зору різних учасників великого проєкту.
Я задав однакові питання різним учасникам, щоб зрозуміти їхню перспективу.

## Питання

1. Що, на твою думку, пішло не так?
2. Що можна було зробити краще?
3. Чого тобі бракувало?
4. Що було добре?
